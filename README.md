# Project Brahmagupta

Project Brahmagupta is a collection of problems in mathematics and computation, inspired by the Indian tradition of ganita and by mathematicians such as Brahmagupta, Bhāskara II, and Mahāvīra.

The collection is still growing. If you have a problem drawn from the Sanskrit mathematical tradition, or one written in a similar spirit, you may send it for consideration.
